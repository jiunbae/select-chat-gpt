# SelectChatGPT - Chrome 웹 스토어 설명

## 짧은 설명 (132자 이내)
ChatGPT 대화에서 원하는 메시지만 선택하여 공유하세요. 링크, 마크다운, PNG, PDF로 내보내기 가능.

## 상세 설명

### 개요
SelectChatGPT는 ChatGPT 대화를 선택적으로 공유할 수 있게 해주는 확장 프로그램입니다. 전체 대화를 공유하는 대신, 원하는 메시지만 골라서 공유할 수 있습니다.

### 주요 기능

**메시지 선택**
- ChatGPT 공유 대화에서 특정 메시지만 선택
- 체크박스로 포함할 메시지를 정확하게 선택
- 한 번의 클릭으로 전체 선택 / 전체 해제
- 역할별(사용자/어시스턴트) 검색 및 필터링

**다양한 내보내기 옵션**
- **공유 링크**: 선택한 메시지만 포함된 공유 링크 생성
- **마크다운**: .md 파일로 복사 또는 다운로드
- **PNG 이미지**: 소셜 미디어 공유용 고품질 이미지
- **PDF 문서**: 인쇄 및 보관용 다중 페이지 문서

**PDF 커스터마이징**
- 페이지 크기 선택 (A4, Letter 등)
- 여백 및 레이아웃 조정
- 머리글/바닥글 추가 (날짜, 제목, 페이지 번호)
- 글꼴 크기, 줄 높이, 자간 사용자 지정
- 필요시 사용자 질문이나 코드 블록 숨기기

**개인정보 보호**
- ChatGPT 공유 대화 페이지에서만 작동
- 대화 내용 추적 없음
- 가능한 경우 로컬에서 데이터 처리

### 사용 방법
1. ChatGPT 공유 대화를 엽니다 (chatgpt.com/share/...)
2. 공유할 메시지 옆의 체크박스를 클릭합니다
3. "공유 링크 생성"을 클릭하거나 내보내기 형식을 선택합니다
4. 링크를 공유하거나 파일을 다운로드합니다

### 지원 언어
- English (영어)
- 한국어
- 中文 (중국어)

### 권한 설명
- **activeTab**: 선택 UI를 표시하기 위해 현재 ChatGPT 공유 페이지에 접근
- **storage**: 내보내기 환경설정을 로컬에 저장
- **clipboardWrite**: 공유 링크와 마크다운을 클립보드에 복사
