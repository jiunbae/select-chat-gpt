# SelectChatGPT - Chrome Web Store Description

## Short Description (132 characters max)
Select and share specific ChatGPT messages. Export conversations as Link, Markdown, PNG, or PDF.

## Detailed Description

### Overview
SelectChatGPT lets you selectively share ChatGPT conversations. Instead of sharing entire conversations, pick only the messages you want to share.

### Key Features

**Message Selection**
- Select specific messages from ChatGPT shared conversations
- Use checkboxes to pick exactly which messages to include
- Select all / Deselect all with one click
- Search and filter messages by role (User/Assistant)

**Multiple Export Options**
- **Share Link**: Create a shareable link with only selected messages
- **Markdown**: Copy or download as .md file
- **PNG Image**: High-quality image for social media sharing
- **PDF Document**: Multi-page document for printing and archiving

**PDF Customization**
- Choose page size (A4, Letter, etc.)
- Adjust margins and layout
- Add headers/footers (date, title, page numbers)
- Customize font size, line height, letter spacing
- Hide user questions or code blocks if needed

**Privacy-Focused**
- Works only on ChatGPT shared conversation pages
- No tracking of your conversations
- Data processed locally when possible

### How to Use
1. Open a ChatGPT shared conversation (chatgpt.com/share/...)
2. Click checkboxes next to messages you want to share
3. Click "Create Share Link" or choose export format
4. Share the link or download the file

### Supported Languages
- English
- Korean (한국어)
- Chinese (中文)

### Permissions Explained
- **activeTab**: Access current ChatGPT share page to display selection UI
- **storage**: Save your export preferences locally
- **clipboardWrite**: Copy share links and markdown to clipboard
